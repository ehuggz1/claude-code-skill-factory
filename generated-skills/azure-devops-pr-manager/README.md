## Features

### 1. Create Pull Requests
- Smart PR descriptions from commits and diffs
- Auto-suggest reviewers from CODEOWNERS and git blame
- Link work items from commit messages
- Support for draft PRs and auto-complete

### 2. Review Pull Requests
- Automated code analysis for security, quality, and performance
- Generate actionable review comments
- Calculate quality scores and risk levels
- Post comments directly to Azure DevOps

### 3. Manage Pull Requests
- Complete/merge with validation
- Abandon with reasons
- Update PR details
- Check policies and merge conflicts

### 4. PR Analytics
- Track cycle time, review time, PR size
- Monitor team performance
- Identify bottlenecks and trends
- Generate comprehensive reports
